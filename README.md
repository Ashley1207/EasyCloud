### 简介
EasyCloud是一个前后端分离的简易文件云平台，支持管理与分享文件。  

### 技术栈
- 后端使用Spring, SpringMVC, Mybatis, MySQL实现,详见 [server/README.md](./server/README.md)
- 前端使用Vue.js, Element UI实现
